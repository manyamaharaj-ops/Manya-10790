RAILWAY TICKET BOOKING SYSTEM

Project Description:-
This project is a simple Railway Ticket Booking System developed in C. It allows users to check seat availability, book-tickets, and cancel booked seats using a menu-driven interface.

Purpose of the Project:-
The purpose of this project is to understand the basic working of a railway reservation system and to practice fundamental concepts of C programming.

Utility of the Project:-
This project is useful for beginners to learn how arrays and functions are used to manage data in a simple reservation system.

Features:-
Seat availability display
Ticket booking
Ticket cancellation
Menu-driven program
Total seats :10

Future Scope:-
This project can be improved by adding features like saving bookings permanently using files, searching for available seats quickly, and adding a simple user login system for more practical use.
